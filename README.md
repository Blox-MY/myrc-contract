# BLOX MYRC contracts

## Setup

1. Copy the ```.env.example``` file to ```.env``` and add the relevant fields.
2. Then run:
```shell
yarn install
npx hardhat compile
```

## Run test

```sh
npx hardhat test test/myrc.test.ts
```
